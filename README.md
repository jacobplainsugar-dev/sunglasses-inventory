# Sunglasses Inventory App

This is a small web app for sunglasses inventory.

It includes:

- sunglasses cards with image, color, size, type, stock left, total sold, and popularity
- search by style, color, size, or Mens/Womens
- best sellers
- add new sunglasses
- quick restock buttons for +1, +5, +10, and +100
- quick sold button for -1
- recent history
- Supabase connection at the bottom of the app

## Supabase Setup

1. Open your Supabase project.
2. Go to SQL Editor.
3. Paste the code from `supabase-schema.sql`.
4. Run it.
5. Refresh this app.

The app uses one table:

- `sunglasses`

The project URL and anon key are already filled into the app defaults.
